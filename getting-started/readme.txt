=======================================================================================
Lovebyte X86 sizecoding starter pack 

This package includes:
- NASM Assembler / NDISASM Disassembler for Windows
- Turbo Debugger for DOS (use TD yourfile.com from dosbox)
- Dosbox emulator
- T.BAT batchfile to assemble your sourcecode (e.g. T minimal)
- RUN.BAT batchfile to start Dosbox

And the following examples:
- Minimal updateloop/framework with a XOR pattern (CCCD constant)
- Alternative minimal Minimal updateloop/framework with a XOR pattern (Division by 320)
- A FPU example using a perspective floor effect
